## 📖 About

**BRUTALGUIDE** is a comprehensive web application designed to help students make informed decisions about engineering education in Maharashtra, India. Whether you're a student who just completed 10th or 12th standard, this guide provides all the information you need about:

- 🎓 **20+ Engineering Branches** with detailed information
- 🏛️ **Top Colleges** in Maharashtra with rankings
- 📊 **Comparison Tables** for easy decision making
- 💼 **Career Paths** and job prospects
- 📝 **Entrance Exams** guidance (JEE, MHT-CET, etc.)
- 📈 **Future Trends** in engineering

## ✨ Features

### 🗂️ Comprehensive Branch Information
Click on any engineering branch to explore:
- **Definition & Overview** - What the branch is all about
- **Core Subjects** - Curriculum details
- **Required Skills** - What you need to succeed
- **Career Opportunities** - Job roles and positions
- **Top Recruiters** - Companies that hire
- **Salary Range** - Expected compensation
- **Top Colleges** - Where to study
- **Certifications** - Additional qualifications
- **Higher Studies** - Post-graduation options

### 📱 Modern UI/UX
- **Responsive Design** - Works on all devices
- **Dark Theme** - Easy on the eyes
- **Smooth Navigation** - Left sidebar with all sections
- **Interactive Cards** - Click for detailed modals
- **PDF Download** - Get the complete guide offline
- **ZIP Download** - Download the entire project source code

### 📊 Sections Included
1. **Introduction** - Overview of engineering in Maharashtra
2. **Career After 10th/12th** - Pathways and options
3. **Classification** - Types of engineering streams
4. **All Branches** - Detailed branch cards
5. **Diploma Engineering** - Polytechnic options
6. **Rankings 2026** - Latest college rankings
7. **Comparison Tables** - Side-by-side analysis
8. **Top Colleges** - Premier institutions
9. **Entrance Exams** - JEE, MHT-CET, GATE info
10. **Future Trends** - Emerging fields
11. **Conclusion** - Final recommendations

## 🛠️ Installation

### Prerequisites
- Node.js 18+ or Bun
- npm, yarn, or bun package manager

### Clone & Setup

```bash
# Clone the repository
git clone https://github.com/brutal-45/BRUTALGUIDE.git

# Navigate to project directory
cd BRUTALGUIDE

# Install dependencies
bun install
# or
npm install

# Start development server
bun run dev
# or
npm run dev
```

### Build for Production

```bash
# Build the application
bun run build

# Start production server
bun run start
```

The application will be available at [Deployed here👋](https://brutalguide.vercel.app/)
